Tab选项卡组件

Tab选项卡，使用Tab方式展示内容

类名

hait.layout.TabLayout

属性

  • options : 保存所有选项卡配置数据
  • style : 当前样式，一般用于设置宽高

事件

  • onchange : 选项变更事件

方法

  • draw() : 直接绘制该对象
  • addTab(options) 添加Tab
    • options : Tab的属性，例如：{text:"欢迎页",url:"welcome.jsp"}
  • getTab(index)：根据下标获取对应的Tab栏的dom对象，例如：tabLayout.getTab(0).src = "http://www.baidu.com";
  • closeTab(key)：删除对应的Tab栏
  • selected(index,isIgnoreOnChange) : 根据index选中项目，isIgnoreOnChange为是否忽略变更事件

代码2：代码编写

	tabLayout = new hait.layout.TabLayout();
	tabLayout.setParam({
		options : [ {
			type : "iframe",
			text : "选项1",
			url : "tablayout_page1.html"
		}, {
			type : "page",
			text : "选项2",
			page: $("#mytest3")
		} ]
	});
	tabLayout.draw();
	$("#mytab1").append(tabLayout.obj);
	function openNewTab() {
		tabLayout.addTab({
			text : "选项3",
			close : true,
			url : "tablayout_page2.html"
		});
	}
