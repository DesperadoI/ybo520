放大镜组件

表单元素，放大镜组件，用于集成各种复杂选项，通过对子页面的编写支持单复选，继承自hait.form.Field元素

类名

hait.form.field.MagnifierField

属性

  • id : 如果没有设置，那么系统将自动生成
  • name : 名称
  • text : 描述名称
  • type : 类别
  • defaultValue : 默认值
  • value : 当前值
  • placeholder : 占位符
  • required : 是否必须
  • disabled : 是否禁用
  • readonly : 是否只读
  • pattern : 正则表达式
  • patternMessage : 当正则表达式没有验证通过时显示的提示
  • style : 最外层的样式设置
  • displayValue : 显示的值
  • displayName : 显示的值字段名称，用于表单load时，自动绑定显示值
  • url : 功能子页面地址
  • dialogTitle : 对话框标题
  • dialogStyle : 对话框样式，一般用于设置宽高

方法

  • draw() : 直接绘制该对象
  • getValue() : 获取值
  • getDisplayValue() : 获得显示的值
  • setValue(val) : 设置值
  • setDisplayValue(displayValue) : 设置显示值
  • setDisabled(isDisabled) : 设置禁用
  • setReadonly(isReadonly) : 设置只读
  • validate() : 验证自身数据的方法，验证是否必须，正则表达式
  • reset() : 重置的基本实现
  • changeURL(url) : 变更内部iframe的地址
  • setDialogTitle(title) : 变更对话框标题

事件

  • ondialogclose(resultValue) : 对话框关闭后的回调事件
