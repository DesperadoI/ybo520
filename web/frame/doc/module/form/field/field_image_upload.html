图片上传选择框

表单元素，图片上传选择框，继承自hait.form.Field元素

类名

hait.form.field.ImageUploadField

属性

  • id : 如果没有设置，那么系统将自动生成
  • name : 名称
  • text : 描述名称
  • type : 类别
  • defaultValue : 默认值
  • value : 当前值
  • placeholder : 占位符
  • required : 是否必须
  • disabled : 是否禁用
  • readonly : 是否只读
  • pattern : 正则表达式
  • patternMessage : 当正则表达式没有验证通过时显示的提示
  • style : 最外层的样式设置
  • limit : 允许上传的文件数量
  • fileName : 文件名称，设置后，上传的文件会以此命名
  • purpose : 图片用途，重要参数，用于确定是否唯一，有效期、存放位置等多种属性

方法

  • draw() : 直接绘制该对象
  • getValue() : 获取值
  • setValue(val) : 设置值
  • setDisabled(isDisabled) : 设置禁用
  • setReadonly(isReadonly) : 设置只读
  • validate() : 验证自身数据的方法，验证是否必须，正则表达式
  • reset() : 重置的基本实现

代码2：代码编写

	var fileField = new hait.form.field.ImageUploadField();
	fileField.limit = 15;
	fileField.fileName = "test";
	fileField.purpose = "CUST_CARD";
	fileField.draw();
	$("#test").append(fileField.obj);
