时间选择框

表单元素，时间选择框，继承自hait.form.Field元素

类名

hait.form.field.DateField

属性

  • id : 如果没有设置，那么系统将自动生成
  • name : 名称
  • text : 描述名称
  • type : 类别
  • defaultValue : 默认值
  • value : 当前值
  • placeholder : 占位符
  • required : 是否必须
  • disabled : 是否禁用
  • readonly : 是否只读
  • pattern : 正则表达式
  • patternMessage : 当正则表达式没有验证通过时显示的提示
  • style : 最外层的样式设置
  • format : 格式化文本yyyy-MM-dd 或者 yyyy-M-d
  • startDate : 可选择的开始日期
  • endDate : 可选择的结束日期

事件

  • onchange(date,prevValue,curValue) :当选择变化的时候触发

方法

  • draw() : 直接绘制该对象
  • getValue() : 获取值
  • setValue(val) : 设置值
  • setDisabled(isDisabled) : 设置禁用
  • setReadonly(isReadonly) : 设置只读
  • validate() : 验证自身数据的方法，验证是否必须，正则表达式
  • reset() : 重置的基本实现

使用示例

代码1：直接在页面上书写元素（下划线改为减号使用）

代码2：代码编写

	var date = new hait.form.field.DateField();
	date.placeholder = "请选择时间...";
	date.startDate = "2016-11-10";
	date.endDate = "2016-11-20";
	date.draw();
	$("#mydate").append(date.obj);
