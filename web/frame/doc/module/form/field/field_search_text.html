查询输入框

表单元素，查询输入框，在输入数据的同时查询后台数据

类名

hait.form.field.SearchTextField

属性

  • id : 如果没有设置，那么系统将自动生成
  • name : 名称
  • text : 描述名称
  • type : 类别
  • defaultValue : 默认值
  • value : 当前值
  • placeholder : 占位符
  • required : 是否必须
  • disabled : 是否禁用
  • readonly : 是否只读
  • pattern : 正则表达式
  • patternMessage : 当正则表达式没有验证通过时显示的提示
  • style : 最外层的样式设置
  • mode : 数据来源模式，包括func,url，默认为func
  • options : 下拉选项值数组
  • server : 目标服务器地址，默认可不设置，可写值：CUSTOMER_SERVER，DRIVER_SERVER
  • funcId : 功能编号
  • params : 参数
  • optionValue : 值列项
  • auto : 是否自动获取数据，默认true
  • search : 查询字段，默认为optionText
  • searchStart : 当输入框内的内容达到此长度时，才进行查询，默认为1

方法

  • draw() : 直接绘制该对象
  • getValue() : 获取值
  • getDisplayValue() : 获取显示的值
  • setValue(val) : 设置值
  • setDisabled(isDisabled) : 设置禁用
  • setReadonly(isReadonly) : 设置只读
  • validate() : 验证自身数据的方法，验证是否必须，正则表达式
  • reset() : 重置的基本实现

事件

  • onchange(input,prevValue,curValue,option) :当选择变化的时候
  • onrefresh() : 完成刷新后执行的事件
  • onload(input) : 当数据获取完毕时执行的事件
  • onparam(input,val) : 当拼装查询参数时执行的事件
