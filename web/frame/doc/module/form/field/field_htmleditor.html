HTML编辑器

表单元素，HTML编辑器，继承自hait.form.Field元素

类名

hait.form.field.EditorField

属性

  • id : 如果没有设置，那么系统将自动生成
  • name : 名称
  • text : 描述名称
  • type : 类别
  • defaultValue : 默认值
  • value : 当前值
  • placeholder : 占位符
  • required : 是否必须
  • disabled : 是否禁用
  • readonly : 是否只读
  • pattern : 正则表达式
  • patternMessage : 当正则表达式没有验证通过时显示的提示
  • style : 最外层的样式设置
  • container : 编辑器生成的容器

方法

  • draw() : 直接绘制该对象
  • getValue() : 获取值
  • setValue(val) : 设置值
  • setDisabled(isDisabled) : 设置禁用
  • setReadonly(isReadonly) : 设置只读
  • validate() : 验证自身数据的方法，验证是否必须，正则表达式
  • reset() : 重置的基本实现
  • insertHtml(val) : 将一段内容追加到编辑器的后面

	var editorField = new hait.form.field.EditorField();
	// 由于编辑器使用ueditor，系统无法控制其的生成，所以需要事先设置容器地址
	editorField.container = $("#test");
	editorField.draw();
