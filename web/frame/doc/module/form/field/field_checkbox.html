复选框Checkbox

表单元素，复选框Checkbox，继承自hait.form.Field元素

类名

hait.form.field.CheckboxField

属性

  • id : 如果没有设置，那么系统将自动生成
  • name : 名称
  • text : 描述名称
  • type : 类别
  • defaultValue : 默认值
  • value : 当前值
  • required : 是否必须
  • disabled : 是否禁用

方法

  • draw() : 直接绘制该对象
  • getValue() : 获取值
  • setValue(val) : 设置值
  • isChecked() ：是否选中
  • setDisabled(isDisabled) : 设置禁用
  • validate() : 验证自身数据的方法，验证是否必须，正则表达式
  • reset() : 重置的基本实现

事件

  • onclick(input, checked) : 点击事件
