表单元素

所有表单元素组件的父类

类名

hait.form.Field

属性

  • name : 名称
  • nameText : 中文名称
  • text : 描述名称
  • type : 类别
  • defaultValue : 默认值
  • value : 当前值
  • placeholder : 占位符
  • required : 是否必须
  • disabled : 是否禁用
  • readonly : 是否只读
  • pattern : 正则表达式
  • patternMessage : 当正则表达式没有验证通过时显示的提示
  • style : 最外层的样式设置

方法

  • validate() : 验证自身数据的方法，验证是否必须，正则表达式
  • setReadonly(isReadonly) : 设置只读的基本实现
  • setDisabled(isDisabled) : 设置禁用的基本实现
  • reset() : 重置的基本实现
