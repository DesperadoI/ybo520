表单

用于将一个表单转换成hait表单，使得拥有一系列功能，并且自动转换其中的表单组件

类名

hait.form.Form

属性

  • server : 目标服务器地址，默认可不设置，可写值：CUSTOMER_SERVER，DRIVER_SERVER

事件

  • onafterload(formData) : load数据完成后执行的
  • onaftersubmit(response) : 提交完成后执行的

方法

  • convert(obj) : 初始化表单及其内部元素
  • load(funcId, params, method) : 根据funcId和参数load表单数据
  • validate() : 启用HTML5的表单验证程序，返回true/false
  • submit(funcId, params, method) : 根据传入的功能号和参数提交到数据层，注：表单内的组件数据会自动提交
  • reset() : 重置表单元素
  • setValue(fieldName, fieldValue) : 设置fieldName元素的值为fieldValue
  • getValue(fieldName) : 获取fieldName元素的值
  • getValue() : 获取表单中全部的值，格式为JSON
  • getElement(fieldName) : 获取fieldName元素的对象
