分组表格

用于生成一个分组表格，继承自hait.grid.Grid

类名

hait.grid.GroupGrid

属性

  • 和表格一致
  • groupField : 分组字段名称，默认为text

事件

  • 和表格一致

方法

  • 和表格一致

使用示例

代码1：直接在页面上书写元素（下划线改为减号使用）
