全局方法

  • inherit(Class) 继承
  • request(ajaxParam) 统一的请求数据的方法
    • 例如：request({
      url:'data.do', //提交url, 默认'data.do'
      async:true, //是否异步，默认true
      loading:true, //是否存在提示框，默认true
      global:{}, //请求顶级参数
      data:[], //提交数据，可多个
      func:function(data){}, //回调函数
      })
  • hait.namespace(fullNS) 创建命名空间
    • 例如：hait.namespace("hait.form");
  • hait.register(comp) 注册组件，系统立即载入该组件，如果该组件已经在缓存中存在，那么就直接读取缓存中的信息
    • 例如：hait.register("hait.form.field.SelectField");
  • hait.resource(path) 获取JS资源文件，如果该文件存在于缓存中，那么直接从缓存读取
    • 例如：hait.resource("../../../plugins/wdtree/jquery.tree.js");
  • hait.getCompById(id) 根据编号获取已经生成的hait组件
  • hait.getCompByName(name) 根据名称获取已经生成的hait组件
