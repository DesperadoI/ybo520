常用工具方法

  • hait.analyzeURL(url) : 解析url中参数，将之封装为json
  • hait.parseParam(param, type) : 转换字符串中的param部分的内容
  • String.replaceCharAt (idx, str) : 根据下标更新字符串
  • String.trim() : 去掉字符串两边的空格
  • String.deleteLastComma() : 删除最后的逗号
  • String.replaceAll(reallyDo, replaceWith) : 使用reallyDo替换字符串中的replaceWith
  • Date.format(fmt) : 格式化诶起

hait.util.Map

  • put(key, value) : 添加对象到Map
  • get(key) : 根据key从Map对象中获取数据
  • remove(ley) : 根据key删除value
  • size() : 得到当前Map对象的长度
  • containsKey(key) : 判断是否包含key
  • keys() : 得到Map对象的键集
  • isEmpty() : 判断Map对象是否为空
  • each(callback) : 对Map对象进行循环
  • hait.util.Map.isMap(map) : 判断某对象是否为Map对象
