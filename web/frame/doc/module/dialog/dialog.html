对话框

用于生成一个对话框，内部可以嵌任何东西

类名

hait.dialog.Dialog

属性

  • id : 对话框编号
  • title : 对话框标题
  • style : 附加样式
  • body : 对话框内容，可以是任何对象
  • isMask : 是否存在遮罩层
  • buttons : 对话框按钮栏，可以没有

方法

  • draw() : 绘制对象
  • show() : 展示对话框
  • hide() : 隐藏对话框
  • isDisplay() : 返回是否展示着
  • remove() : 删除该对话框
  • setTitle(title) : 设置对话框标题

事件

  • onafterclose : 关闭按钮点击后调用的事件
