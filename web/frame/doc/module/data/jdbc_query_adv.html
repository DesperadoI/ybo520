JDBC关联查询

JDBC关联查询，XML配置中的参数项，可以是一个外部数据，这点主要用于联表查询

操作示例

元数据配置：

out属性

  • name : 和数据库对应的字段名称
  • type : 和数据对应的类别
  • foreign : 外键表名
  • key : 本表和外键表对应的字段
  • fkey : 外键表和本表对应的字段，如果为空，那么值为key
  • column : 外键表要显示的字段

查询数据操作： 

		request({
			data : [ {
				funcId : "hex_doc_sys_dict_items_query",
				dict_id : "5"
			} ],
			func : function(data) {
				alert(JSON.stringify(data));
			}
		});
