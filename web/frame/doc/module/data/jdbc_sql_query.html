JDBC自定义SQL查询

JDBC自定义SQL查询，在XML配置中，可以使用自定义SQL方式

操作示例

元数据配置：

		
			
			
			select a.*,b.dict_name
			from t_sys_dict_items a
			inner join t_sys_dicts b on a.dict_id = b.dict_id
			where a.dict_id = #dict_id

注：此时，传入的参数会自动替换SQL语句中的#开头的部分，例如上面的#dict_id

查询数据操作： 

		request({
			data : [ {
				funcId : "hex_doc_sys_dict_items_sql_query",
				dict_id : "5",
			} ],
			func : function(data) {
				alert(JSON.stringify(data));
			}
		});
